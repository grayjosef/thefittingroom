Accessibility Statement

Effective: May 4, 2026 · Last updated: May 4, 2026

We want every client to be able to read about the Studio, book a consultation, and find their way to us. This page describes what we’ve done and how to ask for help.

01Commitment

We aim to meet the Web Content Accessibility Guidelines (WCAG) 2.2 at level AA across the website and booking flow. Accessibility is reviewed each time we update the site.

02Features in place

  • Semantic landmarks and heading structure throughout
  • Visible focus rings on all interactive elements
  • Color contrast tested against WCAG AA across body and accent text
  • Keyboard navigation support across the booking flow
  • Form fields labelled with descriptive text
  • Reduced-motion respected for animations
  • Responsive type scale on tablet and mobile

03Studio access

The Stevens Point studio is a private space. If you have specific access needs — mobility, sensory, or otherwise — please let us know in advance and Catherine will arrange the appointment to accommodate you.

04Feedback

If anything on this site is hard to use, please tell us. We treat accessibility issues as bugs and aim to resolve them promptly.

Email inquiry@thefittingroom-gh.com with the page, the issue, and the device or assistive technology you were using.
