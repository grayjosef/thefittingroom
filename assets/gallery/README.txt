Gallery photos go in this folder.

To add photos to the website:

1. Drop your photo files (JPG or PNG) here in this folder.
   Examples:
     bustle-after.jpg
     hem-detail-1.jpg
     gown-finished.jpg

2. Open the file: components/gallery-items.js

3. Add an entry for each photo (the file shows examples).
   - For a single photo:    type: "single"
   - For before/after pair: type: "pair"

4. Save, commit, and push. The website updates automatically.

The gallery section auto-hides until at least 4 items are configured.
